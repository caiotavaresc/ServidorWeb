Esta pasta contém, além deste manual e do relatório, dois diretórios:

Codigo -> Contém os arquivos .java (arquivos de codigo) referentes à implementação dessa entrega
Classes Compiladas -> Contém os arquivos .class (compilados) referentes à implementação dessa entrega
	Classes Compliadas/config -> Contém os arquivos de configuração intrínsecos ao servidor e o arquivo de log

As três classes pertencem ao pacote servidorweb. Para executar a aplicação via terminal:
- Direcionar até o diretório das classes compiladas, por exemplo: C:\Users\ctcca\Documents\GitHub\ServidorWeb\9004525_8921840_EntregaA\Classes Compiladas
- Escrever: java servidorweb.ServidorWeb
- A partir daí a aplicação já estará rodando, basta fazer os testes

ATENÇÃO: O diretório raiz do servidor web é a pasta "Classes Compiladas", é por isso que o diretório "config" e o arquivo "index.html" estão nessa pasta, isto significa que qualquer arquivo que você deseja que seja encontrado em localhost:6789/ deve ser acessível a partir desta pasta. O arquivo best.jpg é um exemplo disto: como ele está na pasta "Classes Compiladas", o acesso em localhost:6789/best.jpg retorna a imagem.
